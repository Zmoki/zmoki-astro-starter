This post exists to show the features wired into the Markdown/MDX pipeline. Delete it once you've seen them, or keep it as a reference.

## Headings and text

Regular paragraphs, **bold**, _italic_, and `inline code` all work. Headings get anchor links automatically.

## Links

Internal links stay on the site. [External links](https://example.com) open in a new tab and are styled differently. So are [resource links](/resources/example-resource/).

## Code blocks

Fenced code blocks are syntax-highlighted and get a copy button:

```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```

## Definition lists

Terms get anchor `id`s so you can link straight to them:

term
: a word or phrase being defined

design token
: a named value, like a color, defined once and reused everywhere
